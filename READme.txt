# Eazy Mechanics Website

## Overview
Eazy Mechanics is a professional automotive service website for a premier car and motorcycle maintenance service located in Secunda, Mpumalanga, South Africa.

## Features
- Responsive design that works on mobile, tablet, and desktop devices
- Modern, clean UI with automotive-themed visuals
- Interactive navigation with smooth scrolling
- Service showcase with car and motorcycle maintenance options
- Contact form with validation
- Business information with hours of operation

## Technical Stack
- HTML5
- CSS3 (with custom styles)
- Vanilla JavaScript
- Font Awesome icons
- Unsplash images for automotive visuals
